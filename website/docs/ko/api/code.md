# HTTP 상태 코드

|  상태 코드             |  설명                      | 해결방법                 |
| -------------------- | ------------------------- | ---------------------- |
|  200                 |  정상처리                   |  필요없음                |
|  400                 |  Bad Request              | 요청 정보 수정            |
|  401                 |  Unauthorized             | 인증 정보 수정            |
|  500                 |  Internal Server Error    | 잠시후 요청              |
|  503                 |  Service Unavailable      | 메인터넌스후 요청          |
