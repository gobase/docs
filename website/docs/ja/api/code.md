# HTTP ステータスコード

|  ステータスコード       |  説明               | 原因                    |
| -------------------- | ------------------- | ---------------------- |
|  200                 |  正常処理            |  処理が正しく行われた      |
|  401                 |  認証が不正           | 認証情報が間違った        |
|  403                 |  アクセス不可         | APIパスが間違ってた       |
|  500                 |  サーバー内部エラー    | しばらく経ってからアクセス  |
|  503                 |  メンテナンス         | メンテナンス完了を待つ     |
