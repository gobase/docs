# HTTP状态码

|  状态码               |  说明                      |  解决方法                   |
| -------------------- | ------------------------- | -------------------------- |
|  200                 |  OK                       |  不需要                     |
|  400                 |  Bad Request              | 查看请求内容                 |
|  401                 |  Unauthorized             | 查看署名方法                 |
|  500                 |  Internal Server Error    | 稍后再请求                  |
|  503                 |  Service Unavailable      | 等维修过后再请求             |
